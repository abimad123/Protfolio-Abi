#  Portfolio – Abi

A modern, responsive portfolio website showcasing my **UI/UX design** and **full-stack web development** projects.  
Built using **HTML5**, **CSS3**, and **JavaScript**, the site highlights creativity, clean code, and seamless user experience.

---

##  Features
-  **Modern UI/UX:** Minimal, responsive, and fully optimized for all devices.  
-  **Interactive Elements:** Smooth animations and transitions powered by JavaScript and Three.js.  
-  **Projects Showcase:** Highlighting real-world applications like Kirpa Home Solutions and My Amazspace.  
-  **Contact Form Integration:** Functional Formspree-based form for direct communication.  
-  **Scalable Design:** Modular and easy to update for adding future projects or case studies.

---

##  Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript  
- **Design Tools:** Figma, Canva  
- **Libraries & Integrations:** Three.js, Formspree  
- **Deployment:** Vercel
